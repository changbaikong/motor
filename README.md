# 四电机 PID 调速（蓝牙控制）

## 文件结构

```
motor.ino           # 主程序：初始化 + 主循环（仅 ~70 行）
encoder.h/.cpp      # 编码器：引脚定义、中断初始化、脉冲计数、定时器 RPM/距离计算
pidctrl.h/.cpp      # PID 控制器：KP/KI 参数、误差计算、PWM 更新
btcontrol.h/.cpp    # 蓝牙+电机控制：电机对象、蓝牙命令读取、电机驱动
motor.h/.cpp        # 电机驱动：正转/反转/停止/刹车
Blue-smooth.h/.cpp  # 蓝牙通信底层
```

## 电机 & 编码器映射

| 电机 | 位置 | 对象 | 编码器引脚 | KP | KI |
|------|------|------|-----------|------|------|
| A | 后左 | `motorA(13, 26, 28)` | 2, 3 | 0.15 | 0.06 |
| B | 后右 | `motorB(10, 24, 22)` | 19, 18 | 0.15 | 0.06 |
| C | 前左 | `motorC(9, 30, 32)` | 52, 53 | 0.18 | 0.06 |
| D | 前右 | `motorD(8, 34, 36)` | 50, 51 | 0.18 | 0.06 |

## 引脚总览

### 电机引脚
| 电机 | EN (PWM) | IN1 | IN2 |
|------|----------|-----|-----|
| A | 13 | 26 | 28 |
| B | 10 | 24 | 22 |
| C | 9 | 30 | 32 |
| D | 8 | 34 | 36 |

### 编码器引脚
| 编码器 | A 相 | B 相 | 中断方式 |
|--------|------|------|----------|
| A（后左）| 2 | 3 | INT4 / INT5 |
| B（后右）| 19 | 18 | INT2 / INT3 |
| C（前左）| 52 | 53 | PCINT0（PORTB） |
| D（前右）| 50 | 51 | PCINT0（PORTB） |

- A/B 编码器使用 Mega 2560 硬件外部中断引脚（2, 3, 18, 19）
- C/D 编码器共用 PCINT0，监听 PORTB 引脚变化（50~53）
- SDA(20)/SCL(21) 留给 I2C 使用

## 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `target_rpm` | 150 | 目标转速 |
| `ENC_PPR` | 1040 | 每圈脉冲数（线数 × 减速比） |
| `WHEEL_CIRC_MM` | 151 | 轮子周长（mm） |
| `TIMER_US` | 50000 | 定时器间隔（50ms） |
| PID delta 限幅 | ±40 | delta 变化量上限 |
| 积分限幅 | ±60 | iTerm 上限 |
| PWM 范围 | 15~255 | 输出限幅 |

## 蓝牙控制

| 按键 | 动作 |
|------|------|
| `w` | 四电机正转 |
| `s` | 四电机反转 |
| `x` | 四电机停止 |

## 串口输出

- **Serial（绘图器）**：4 条速度曲线 — `ASpeed,BSpeed,CSpeed,DSpeed`
- **Serial2（调试）**：每 500ms 输出四电机 RPM + 距离
